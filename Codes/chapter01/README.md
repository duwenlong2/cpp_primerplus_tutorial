## 代码示例

hello.cpp:
```cpp
#include <iostream>
int main() {
    std::cout << "Hello, C++!" << std::endl;
    return 0;
}
```

## 编译与运行

在终端输入：
```
g++ hello.cpp -o hello.exe
./hello.exe
```
