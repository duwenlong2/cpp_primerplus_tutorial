 计算机内存的基本单元是位(bit)。可以将位看作电子开关，可以开。可以关。关标志0，开表示1.8位的内存块可以设置出256种不同的组合。因为每一种组合有2种设置。8位的总组合数位2*2*2*2*2*2*2*2 即256.因此8位单元可以表示0~255 或者-128到127，每增加一位，组合数便加倍。16位是65536，32位是4294672296个不同的值，64位是18446744073709551616个不同的值。unsigned long存储不了地球上当前人数和银河系的星星数，而long long能够。
